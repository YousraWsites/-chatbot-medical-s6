# Skills — SAE Chatbot LLM S6

## Développement Full Stack
- Concevoir et développer une interface utilisateur (Frontend React JS)
- Développer une API REST (Backend FastAPI)
- Assurer la communication frontend ↔ backend
- Gérer l'authentification et les sessions utilisateurs

## Intelligence Artificielle & LLM
- Intégrer un modèle de langage (LLM) via API ou local
- Comparer plusieurs LLMs : Mistral, DeepSeek, Ollama
- Comprendre le fonctionnement interne d'un LLM (tokenisation, embeddings, génération)
- Appliquer les méthodes de conception vues en cours (Mr FAYE & Mme AZZAG)

## RAG (Retrieval-Augmented Generation)
- Implémenter un pipeline RAG avec LangChain
- Tester et comparer des encodeurs (modèles d'embeddings)
- Indexer des documents médicaux dans une base vectorielle (ChromaDB)
- Améliorer la pertinence des réponses et limiter les hallucinations

## Base de données
- Modéliser et gérer une base de données (SQLAlchemy + SQLite)
- Stocker les utilisateurs, sessions et historiques de conversations

## Recherche web (Bonus)
- Intégrer DuckDuckGo comme moteur de recherche externe
- Enrichir les réponses du chatbot avec des résultats web en temps réel

## Déploiement
- Déployer une application web full stack en ligne (Render / Railway)
- Gérer les variables d'environnement et la configuration de production
